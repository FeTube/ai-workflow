inconsistent_interaction
Naming mismatch: PRD references 'Screen/ErrorState' and 'H5/ErrorState', but design files use 'Screen/LoginError' and 'H5/LoginError'. Recommend aligning to design naming ('LoginError') and updating PRD.
警告
extra_feature
Design includes 'Screen/Login-Remembered' page (a returning-user variant showing avatar, name, and password-only login) which is not described as a separate page in the PRD. PRD section 3.6 mentions 'Remember me' functionality but does not define this UI variant. Recommend adding this page to PRD or confirming it's part of F-AUTH-014.
警告
inconsistent_interaction
ForgotPassword button text mismatch: design shows 'Send reset link' but PRD specifies sending a verification code (section 3.4: 'Send Reset Code'). The actual mechanism is code-based, not link-based. Design button text should be updated to 'Send reset code' for consistency.
警告
缺失页面
H5 端缺少 Login-Remembered 变体页面。PC 端有 Screen/Login-Remembered 但 H5 没有对应的 H5/Login-Remembered。如果确认 Login-Remembered 为正式需求，需补齐 H5 端设计稿。
提示
模糊需求
PRD section 2.5 requires password strength indicator (weak/medium/strong with color differentiation), but neither PC nor H5 design files include a password strength meter component. Need to confirm whether this is deferred or should be added to designs.
警告
模糊需求

提示
模糊需求
Design includes country-selector component (flag + country name + chevron) on Register and Login pages (both PC and H5), but PRD does not mention country/locale selection functionality. Need to clarify whether this is a future i18n feature or purely decorative.
