如何判断大小？
小需求：改 1-3 文件 / 单端 / 无 DB 变更 / 无 API 变更 / <30 min
大需求：改 5+ 文件 / 跨端 / DB 迁移 / 新增 API / 认证变更 / >1h
不确定：让 Superpowers brainstorming 帮你判断——它在澄清阶段自动评估复杂度，超出简单修复范围会主动建议进入完整流程。
