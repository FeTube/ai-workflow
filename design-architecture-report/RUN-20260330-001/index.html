⚡缓存模型 — Redis Key（9）
auth:reg:{email}
注册暂存（密码哈希 + 协议同意）
类型: string (JSON) TTL: 15 min
auth:code:{email}:{purpose}
验证码，purpose = register | reset_password
类型: string TTL: 10 min
auth:code:fail:{email}:{purpose}
验证码错误次数
类型: string (counter) TTL: 10 min
auth:rate:email:{email}
发信限频计数器
类型: string (counter) TTL: 1 hour
auth:rate:ip:{ip}
IP 注册/发信限频
类型: string (counter) TTL: 1 hour
auth:rate:device:{fp}
设备注册限频
类型: string (counter) TTL: 1 hour
auth:session:{hash}
会话热数据（user_id, device, remembered）
类型: string (JSON) TTL: 同 refresh_token
auth:lock:{userId}
登录锁定标记
类型: string TTL: 30 min
auth:consent:version
最新协议版本号缓存
类型: string (JSON) TTL: 1 hour
